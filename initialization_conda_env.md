# Conda Cheetsheet

1. Create new env

    ```conda create -n {envname}```

2. Check existing envs

    ```conda info --envs```

3. Activate env

    ```conda activate {envname}```

4. Deactivate env

    ```conda deactivate```

5. Delete env

    ```conda remove --name {envname} --all```

6. Install & Update new packages

    ```conda install {package_name}```

    ```conda update {package_name} --all```
